FAQ

What is the Document Object Model (DOM)?
DOM is a tree like structure, which allows javascript to interact with HTML and also manipulate HTML and CSS within the javascript.
How do you select an element from the DOM?
There are three ways to select an element from the DOM. One is directly through tag name. But this is not a good way to interact, because this select all the element using the selected tagname. Another one is, using the class name of the elements available in the HTML file. The most convenient one is using a element's ID. ID is unique and can't be use in multiple elements. Through an elements ID, I can select a specific element.
What is event delegation in the context of the DOM, and why is it useful?
Event Delegation is the technique, by which an event handler is used in the parent element and it can catch up all the child elements under that parent. It is useful, by using in the parent we don't need to repeat event handler.
How do you manipulate an element's attributes and styles using the DOM?
I can manipulate an element's attribute by using set or remove attribute. In the DOM I also can manipulate styles of the HTML bt using .style and then which style I want to manipulate.
